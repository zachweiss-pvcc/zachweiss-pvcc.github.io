Term Definition
machine language The lowest-level programming language, consisting of binary code that the computer's central processing unit (CPU) can directly execute.
debugging The process of identifying and fixing errors (bugs) in a computer program.
assembler A program that translates assembly language code into machine code.
assembly language A low-level programming language that uses symbolic code representing the basic operations of a computer's central processing unit (CPU).
translator A program that converts code from one programming language into another.
compiler A type of translator that translates the entire source code of a program into machine code before execution.
interpreter A type of translator that translates and executes one line of code at a time.
cross-platform software Software that can run on multiple operating systems or platforms without modification.
natural language A human language, such as English, Spanish, or Chinese, used for communication between people.
formal language A language designed for specific applications, often with precise rules and syntax, such as programming languages and mathematical notation.
machine independence The ability of a program to run on different types of computers without modification.
high-level programming language A programming language with a high level of abstraction from the computer's hardware, making it easier for programmers to write code.
object-oriented programming (OOP) A programming paradigm that uses objects and classes for organizing and structuring code.
scripting language A programming language designed for automating the execution of tasks, often in the context of a larger program.
data types: The classification or categorization of data items. Common data types include integer, float, and string.
Integer A data type that represents whole numbers without any decimal points.
Float A data type that represents numbers with decimal points or fractional parts.
string A data type that represents a sequence of characters, typically used for text.
primitive data type A basic data type that is not composed of other data types. Examples include integers and floats.
data structure A way of organizing and storing data to perform operations efficiently.
Python collection data structures Data structures provided by the Python programming language for storing and organizing collections of data.
List A collection data type in Python that is ordered and mutable, allowing the storage of multiple items.
Tuple A collection data type in Python that is ordered and immutable, meaning its elements cannot be changed after creation.
Set A collection data type in Python that is unordered and contains unique elements.
dictionary A collection data type in Python that consists of key-value pairs, allowing efficient retrieval of values based on their associated keys.
function A reusable block of code that performs a specific task, often with parameters and return values.
local variable A variable declared within a specific function or block of code, accessible only within that scope.
global variable A variable declared at the top level of a program, accessible throughout the entire program.
source program The human-readable code written by a programmer in a high-level programming language.
object program The machine code generated by a compiler or assembler from the source program.

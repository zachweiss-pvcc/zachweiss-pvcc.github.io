GitHub Q & A

Question Answer
What is a version control system? An application that tracks modifications to a file or group of files over time so that you can retrieve particular versions at a later time is called a version control system (VCS).
What is Git? With the help of the distributed version control system Git, several developers can collaborate on a project at once. It monitors source code modifications made throughout software development.
What is GitHub? GitHub is an online platform that offers Git repository hosting. On top of Git, it offers additional tools, a web-based interface, and collaborative capabilities.
What is the difference between Git and GitHub? Git is a version control system, whereas GitHub is a platform based on Git technology that offers extra features including a web-based graphical interface, collaboration capabilities, and more. This is the primary distinction between Git and GitHub.
Who started GitHub and when was it started? In 2008, Scott Chacon, Tom Preston-Werner, PJ Hyett, and Chris Wanstrath founded GitHub.
What company owns it now? In 2018, Microsoft purchased GitHub.
How much does a GitHub account cost? For individuals and companies, there are premium options with extra capabilities available, but a basic GitHub account is free.

Git/GitHub Terms

  • Repository: A location where all of the files for your project, together with its version history, are stored.
  • Commit: A snapshot of the repository's modifications.
  • Fork: A replica of a repository that lets you freely test modifications without impacting the source code.
  • Push: Transferring updated files to a remote repository.
  • Pull Requests: A user-submitted suggested modification to a repository that is either approved or rejected by the repository's partners.
  • Workflows: You can configure automated procedures in your repository.
  • Issues: A forum to talk about project ideas, tasks, improvements, and bugs.
  • Raw Button: Lets you see a file's unprocessed content.
  • Blame Button: Indicates who changed each line in a file most recently.

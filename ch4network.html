Term Definition
Network Classifications A network can be classified into different types based on its geographical scope:
  • LAN (Local Area Network): A network that is limited to a small geographic area, such as a single building or a campus.
  • MAN (Metropolitan Area Network): A network that covers a larger geographic area, typically a city or a large campus.
  • WAN (Wide Area Network): A network that spans a large geographical area, often connecting multiple cities or countries.
Network Models Different network models provide frameworks for communication and data exchange:
  • Bus Network: A network topology where all devices share a common communication line or bus.
  • Star Network: A network topology where all devices are connected to a central hub or switch.
Protocols Protocols govern the rules for communication between devices:
  • CSMA/CD Ethernet: A protocol used in Ethernet networks, where devices listen for a clear channel before transmitting.
  • CSMA/CA Wifi: A protocol used in wireless networks, where devices avoid collisions by using a carrier-sensing mechanism.
Combining Networks Various devices and concepts contribute to combining and managing networks:
  • Repeater: Extends the range of a network by amplifying and retransmitting signals.
  • Bridge: Connects and filters traffic between different network segments.
  • Switch: Manages data traffic efficiently within a network.
  • Router: Connects different networks and directs data between them.
  • Gateway: Connects different networks using different communication protocols.
Client/Server Model A network architecture where clients request services from servers.
  • P2P: Peer-to-Peer: A network where devices communicate directly without a central server.
  • Distributed System: A network of independent computers that work together to achieve a common goal.
  • Cloud Computing: A model for delivering computing services over the internet.
Internet Addressing Internet addressing involves various components and tiers:
  • Tier I: The highest level of the Internet hierarchy, comprising the most significant and globally distributed internet service providers (ISPs).
  • Tier 2: Regional ISPs that connect to Tier 1 ISPs, serving a more localized geographic area.
  • Tier 3: Local ISPs that connect to higher-tier ISPs, providing internet services to smaller regions or communities.
IP Address An IP address is a numerical label assigned to each device participating in a computer network that uses the Internet Protocol for communication. It serves as an identifier for locating and addressing devices on the network.
  • ICANN - Internet Corporation for Assigned Names and Numbers: The organization responsible for coordinating the maintenance and procedures of several databases related to the namespaces and numerical spaces of the Internet.
  • DNS - Domain Name System: A hierarchical and decentralized system that translates human-readable domain names into IP addresses.
  • TLD: Top-Level Domain: The highest level in the hierarchical Domain Name System, often representing the type or purpose of the domain.
  • Country-code TLDS: Specific top-level domains assigned to individual countries or territories.
Internet Protocols Internet protocols are standards that define the rules for communication between devices on a network. Here are some common Internet protocols:
  • FTP - File Transfer Protocol: A standard network protocol used to transfer files from one host to another over a TCP-based network, such as the Internet.
  • HTTP - Hypertext Transfer Protocol: The foundation of any data exchange on the Web, used for transmitting hypertext in the form of web pages.
  • SMTP - Simple Mail Transfer Protocol: A protocol for email transmission, allowing electronic mail to be sent and received over the Internet.
World Wide Web The World Wide Web (WWW) is an information space where documents and other web resources are identified by Uniform Resource Locators (URLs) and can be accessed via the Internet. It encompasses a vast array of interconnected documents, images, videos, and more, providing a multimedia experience for users.
  • URL - Uniform Resource Locator: A reference or address used to access resources on the Internet. It typically includes the protocol (e.g., HTTP), domain name, and path to the resource.
  • Client-side Activities: Activities performed on the user's device, typically using web browsers and scripting languages like JavaScript, to enhance the user experience without involving the server.
  • Server-side Activities: Activities performed on the web server to process requests, manage data, and generate dynamic content for the user.
Internet Four-Layer Software The Internet Four-Layer Software model represents the layers involved in communication and data exchange over the Internet. These layers are:
  • Application Layer: The top layer responsible for providing network services directly to end-users or applications. It includes protocols for tasks like email, file transfer, and web browsing.
  • Transport Layer: Manages end-to-end communication and ensures the reliable and orderly delivery of data between devices. Key protocols include TCP (Transmission Control Protocol) and UDP (User Datagram Protocol).
  • Network Layer: Focuses on routing data packets between devices in different networks. Key protocols include IP (Internet Protocol).
  • Link Layer: Handles the physical connection between devices on the same network segment. It includes protocols for data link and physical layers.
Internet Characteristics The Internet possesses specific characteristics that contribute to its functionality and reliability:
  • Packet-Switching Network: A network architecture where data is divided into packets for efficient transmission. Each packet can take a different route to reach its destination, enhancing network efficiency.
  • Fault Tolerant: The ability of the Internet to continue functioning even in the presence of hardware or software failures. Redundant pathways and failover mechanisms contribute to fault tolerance.
  • Redundant: The presence of backup systems, pathways, or resources to ensure continuous operation and minimize the impact of failures or disruptions.
Cybersecurity Threats Various types of cybersecurity threats pose risks to computer systems and networks:
  • Malware: A broad term for malicious software designed to harm or exploit computer systems, including viruses, worms, trojan horses, and spyware.
  • Virus: A type of malware that attaches itself to legitimate programs and spreads when the infected program is executed.
  • Worm: Self-replicating malware that spreads across networks without user intervention.
  • Trojan Horse: Malicious software disguised as legitimate software to deceive users and gain unauthorized access to systems.
  • Spyware: Software that secretly monitors and collects user information without their knowledge or consent.
  • Phishing: A fraudulent attempt to obtain sensitive information by posing as a trustworthy entity, often through deceptive emails or websites.
  • DDoS Attack (Distributed Denial of Service): An attack that floods a network or service with traffic, causing it to become unavailable to users.
Cybersecurity Protections Various cybersecurity measures provide protection against threats and vulnerabilities:
  • Firewall: A network security device that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
  • Spam Filter: Software or hardware designed to identify and block or filter out unwanted and unsolicited email, commonly known as spam.
  • Proxy Server: An intermediary server that acts as a gateway between a user's device and the internet, enhancing security and privacy by masking the user's IP address.
  • Antivirus Software: Software designed to detect, prevent, and remove malicious software, including viruses, worms, and trojan horses.
  • Cryptography: The practice and study of techniques for secure communication in the presence of third parties. It involves encoding information to protect its confidentiality and integrity.
  • Public Key Encryption: An encryption method that uses a pair of keys, a public key for encryption and a private key for decryption, enhancing security in data transmission and communication.

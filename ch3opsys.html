Term Definition
Operating System An operating system is a software component that acts as an intermediary between computer hardware and the computer user...
Job Queue A job queue is a sequence of jobs (programs or tasks) waiting to be processed by a computer system...
FIFO (First In, First Out) FIFO is a method of organizing and manipulating data structures, where the first element added is the first one to be removed...
Interactive Processing Interactive processing refers to a mode of computing where the user and the computer interact in real-time...
Time-Sharing Time-sharing is a technique in which multiple users or processes share the computing resources of a system concurrently...
Multiprogramming Multiprogramming is a computing technique where multiple programs are executed concurrently by a computer system...
Application Software Application software refers to computer programs designed to perform specific tasks or applications for end-users...
System Software System software is a type of software that provides a platform for other software to run...
Utility Software Utility software is a type of system software designed to perform specific tasks related to system management, maintenance, and optimization...
Graphical User Interface (GUI) A Graphical User Interface is a type of user interface that uses graphical elements such as icons, windows, and menus to enable user interaction...
Shell A shell is a command-line interface that allows users to interact with the operating system by entering commands...
Kernel The kernel is the core component of an operating system that manages system resources, including memory, processes, and device communication...
Device Driver A device driver is a software component that allows the operating system to communicate with and control hardware devices...
File Manager A file manager is a software tool that provides a user interface for managing files and directories on a computer...
Memory Manager A memory manager is a component of the operating system responsible for managing the computer's memory...
Virtual Memory Virtual memory is a memory management technique that uses both the computer's physical memory (RAM) and secondary storage...
Bootstrapping Bootstrapping, in the context of computing, refers to the process of loading and initializing the operating system on a computer...
ROM (Read-Only Memory) ROM is a type of non-volatile memory that is used primarily in the startup process of a computer...
Deadlock Deadlock is a situation in computing where two or more processes are unable to proceed because each is waiting for the other to release a resource...
Spooling Spooling (Simultaneous Peripheral Operation On-Line) is a technique used in computing to store data in a temporary queue...
Super User (or Administrator) The super user, also known as an administrator, is a user account with elevated privileges...
Multi-Factor Authentication Multi-Factor Authentication (MFA) is a security process that requires users to provide multiple forms of identification or verification...
